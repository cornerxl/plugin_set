介绍

NoDom插件库

本插件库是在NoDom框架上开发的，包含各种类型的插件。


开始

准备工作

要使用本插件库中的任意插件，您需要引用NoDom框架，然后才能正常使用相关插件。


插件使用示例

引入NoDom.js

        <!--从官网引入NoDom-->

        <script type="text/javascript"
        src="http://www.NoDom.org/download/latest/NoDom.min.js"></script>

        <!--从你的应用引入NoDom,注意修改yourpath为你自己应用的路径-->

        <script type="text/javascript"
        src="yourpath/NoDom.min.js"></script>

HTML代码示例

        <!--x-plugin内写入你要使用的插件名-->
        

        <div x-plugin="paging"></div>
