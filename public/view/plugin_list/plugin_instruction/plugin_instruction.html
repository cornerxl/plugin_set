介绍

NoDom插件库

本插件库是在NoDom框架上开发的，包含缓冲动画、分页、轮播图、日期、进度条等不同类型的插件。


开始

准备工作

要使用本插件库中的任意插件，您需要引用NoDom框架，框架介绍、教程、api等可在NoDom官网进行查看，本文档旨在介绍插件库的使用方法。


插件使用示例

引入NoDom.js

        <!--从官网引入NoDom-->

        <script type="text/javascript"
        src="http://www.NoDom.org/download/latest/NoDom.min.js"></script>

        <!--从你的应用引入NoDom,注意修改yourpath为你自己应用的路径-->

        <script type="text/javascript"
        src="yourpath/NoDom.min.js"></script>

HTML代码示例

        <!--x-plugin内写入你要使用的插件名-->
        

        <div x-plugin="paging"></div>

JS代码示例

        /*插件运行用需依赖父模块，子模块详细教程可查看NoDom框架*/
        

        DD.createModule({
        

        
      name:  'm_plugin_parent',      /*插件父模块名称*/

        
      templateUrl:  '*******',      /*插件父模块模板路径/

        
      requires:  [

        
            {type:'js',path:'*****'},      /*使用插件时，需引入插件所对应的js文件,type为引入文件类型，path为引入文件路径*/

        
            {type:'css',path:'*****'}      /*插件所对应的css文件*/

        
      ],

        
      modules:  [{

        
            name:  "m_example",      /*插件所对应的模块名称*/

        
            el:  '.el-example',      /*插件的渲染容器*/

        
            data: {      /*使用插件需传入的参数，以插件库提供的分页插件为例*/

        
                  page:  1,      /*当前页*/

        
                  row:  10,      /*每页显示的条数*/

        
                  total:  100,      /*总共有多少条数据*/

        
                  to_page:  1      /*跳转到某页*/

        
            }

        
      }]

        });
